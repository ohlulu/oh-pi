# General Search Strategy

## When to Use
Fallback for topics that don't clearly match academic, dev, or comparison categories. Also used when the primary module file is missing.

## Target Sites
- wikipedia.org — overview and structure
- Major news outlets (for current events / industry trends)
- reddit.com — community discussions and real experiences
- news.ycombinator.com — technical community
- Established industry blogs and publications

## Query Patterns
- "{topic} explained"
- "{topic} overview 2025"
- "{topic} site:reddit.com"
- "{topic} site:en.wikipedia.org"
- "{topic} trends 2025 2026"
- "{topic} pros cons"

## Source Evaluation
- **Prefer**: primary sources, Wikipedia (for structure, verify claims independently), established publications
- **Deprioritize**: SEO filler, content farms, undated articles, AI-generated summaries
- **Red flags**: sites with excessive ads, no author attribution, recycled content across multiple domains
